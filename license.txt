Installation Instructions:

1. Un-zip the cardeo-minimal.zip file
2. Upload the entire cardeo-minimal directory to your themes folder
3. Login into your Wordpress Admin
4. Choose Design from the main menu
5. The Cardeo Minimal Theme should appear, select it and hit ok.

--

Changing the Twitter Feed to your account:

1. Open the cardeo-minimal directory and open the footer.php file

2. Find this line of code:

<script type="text/javascript" src="http://twitter.com/statuses/user_timeline/mlambert.json?callback=twitterCallback2&amp;count=1"></script>

3. My Twitter username is mlambert.

4. Change mlambert to whatever your Twitter username is.

5. Save the file and upload it back to the cardeo-minimal theme folder,
   over-writing the old footer.php file on the server.

6. Go to your website and hit refresh, your Twitter account info should now appear.
